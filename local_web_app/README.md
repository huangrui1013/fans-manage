# 主题资料合集本地网页

这是一个仅供个人使用的本地网页版本，不需要后端、不需要登录。

## 打开方式

直接双击打开：

```text
index.html
```

或在浏览器中打开：

```text
F:\code\Think\local_web_app\index.html
```

## 数据保存在哪里

数据保存在当前浏览器的 `localStorage` 中。

注意：

- 不会上传到服务器。
- 不会跨浏览器同步。
- 清理浏览器数据可能会丢失内容。
- 建议在“数据”页面定期导出 JSON 备份。

## 当前功能

- 新建、编辑、删除合集
- 合集组织方式：按类型、按时间、自定义
- 三类条目：链接、来源记录、私密附件记录
- 收件箱：先保存，后归档
- 全部条目搜索和筛选
- 合集详情筛选
- 公开分享预览，私密附件会自动隐藏
- JSON 导入/导出
